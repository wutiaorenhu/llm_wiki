# 胶囊胃镜检查介绍

> 来源: 元宝 (腾讯元宝) · 003-饮食与健康

### **AI**

胶囊胃镜，正式名称是“磁控胶囊胃镜系统”，它让胃部检查变得像吞服一粒小胶囊一样简单。下面这个表格可以让你快速了解它的核心信息，之后我会为你展开更多细节。

| 特性 | 描述 |
| :--- | :--- |
| **工作原理** | 患者吞服内置微型摄像头与无线传输装置的智能胶囊，医生通过体外磁场精确控制其在胃内的运动与姿态，对胃部进行全方位观察并拍摄图像[citation:1][citation:2]。 |
| **主要优点** | 无痛、无创、无需麻醉、无交叉感染风险，检查过程舒适，可观察小肠[citation:1][citation:4][citation:8]。 |
| **核心局限性** | 无法进行组织活检或即时治疗；图像质量易受肠道清洁度影响；费用通常高于传统胃镜[citation:4][citation:12][citation:14]。 |
| **适用人群** | 畏惧或不能耐受传统胃镜（如心脑血管疾病患者、体弱者）者；健康体检及胃癌初筛人群；需监测药物性黏膜损伤或胃部病变随访者[citation:2][citation:7]。 |
| **禁忌人群** | 体内有心脏起搏器等电子设备者；疑似或确诊消化道梗阻、狭窄者；吞咽困难者；孕妇[citation:1][citation:2][citation:5]。 |
| **大致费用** | 以往在3000-5000元，随着技术普及（如AI胶囊内镜的应用），价格已有所下降，部分情况可能降至2000元以下[citation:2][citation:3]。 |
| **医保情况** | 多数地区未纳入医保，但上海、武汉等城市已将其纳入限指征支付（如不明原因消化道出血）[citation:2]。 |

### 🔍 重要提醒：如何选择？

胶囊胃镜和传统胃镜并非“谁更好”的竞争关系，而是“如何选择”的互补关系。你可以参考以下思路：

-   **优先考虑胶囊胃镜的情况**：如果你对传统胃镜极度恐惧、身体状况难以耐受插管麻醉、或需要进行常规的胃部健康筛查，胶囊胃镜是一个非常理想的舒适选择[citation:2][citation:7]。
-   **仍需选择传统胃镜的情况**：当医生高度怀疑存在特定病变，且**很可能需要取样进行病理活检**时（如诊断萎缩性胃炎、确认肿瘤性质），传统胃镜依然是不可替代的金标准[citation:4][citation:12]。此外，如果存在明确的消化道梗阻或狭窄风险，也应避免使用胶囊胃镜[citation:1][citation:13]。

### 💡 检查前后的注意事项

为确保检查顺利和结果准确，需要做好以下准备[citation:1][citation:6][citation:10]：

1.  **检查前**：需要**禁食8-12小时**，让胃部保持空净。有时医生会要求饮用去泡剂，以消除胃内黏液和泡沫，让视野更清晰。
2.  **检查中**：吞服胶囊后，需穿戴一件记录仪背心。检查过程中应避免剧烈运动，并远离强磁场环境（如MRI设备）。
3.  **检查后**：注意观察胶囊是否在**数日內随大便排出**。绝大多数人在24-48小时内可自然排出。如果超过两周仍未排出或有腹痛等不适，应及时就医[citation:11]。
