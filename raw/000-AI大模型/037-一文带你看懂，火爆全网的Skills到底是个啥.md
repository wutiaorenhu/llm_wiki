### **AI**

Anthropic 推出的 **Claude Skills** 是一项旨在让 AI 助手具备模块化、可复用专业能力的功能。你可以将其理解为一套专为 Claude 设计的“可插拔技能包”，它让 Claude 能从“通才”转变为在特定任务上训练有素的“专家”[citation:1][citation:2][citation:7]。

下面这个表格能帮你快速把握其核心概览。

| 特性维度 | 说明 |
| :--- | :--- |
| **核心定位** | 模块化的能力包，教会 Claude “如何”完成一类特定任务，而不仅仅是提供知识[citation:2][citation:3]。 |
| **核心文件** | 每个 Skill 都是一个文件夹，其灵魂是一个名为 `SKILL.md` 的 Markdown 文件，包含技能说明和元数据[citation:1][citation:2]。 |
| **工作机制** | **渐进式披露**：Claude 先看所有技能的简要描述，任务匹配时再按需加载详细指令和资源，高效利用上下文窗口[citation:1][citation:4]。 |
| **核心能力** | 不仅包含文本指令，还能捆绑**可执行脚本**（如 Python）和**资源文件**，处理复杂、流程化的任务[citation:1][citation:7]。 |
| **可组合性** | 单个任务可自动串联多个技能，例如自动调用数据分析技能处理数据，再调用 PPT 技能生成报告[citation:2][citation:7]。 |

### 🛠️ 如何使用Skills

你可以通过多种方式使用Skills，既可以直接利用官方提供的强大技能，也可以创建属于自己的定制化技能。

| 使用方式 | 适用场景 | 操作方法 |
| :--- | :--- | :--- |
| **在 Claude.ai 网页端** | 个人用户处理日常办公任务 | 在 Settings → Capabilities 中启用 Skills 功能。官方预置了处理 PDF、Word、Excel、PPT 等文档的技能，可直接通过对话使用，如“请将这份数据生成一份Excel报表”[citation:2][citation:5]。 |
| **在 Claude Code 中** | 开发者、工程师在编码环境中 | 通过命令添加官方技能市场并安装，例如：`/plugin marketplace add anthropics/skills`，然后安装如 `document-skills` 等技能包[citation:1][citation:2]。 |
| **通过 Claude API** | 开发者将技能集成到应用中 | 使用新的 `/v1/skills` 端点管理技能，在 API 请求中通过 `container` 参数指定要使用的技能ID[citation:4][citation:6]。 |

**创建自定义Skill**：这是Skills功能最强大的地方。你可以将个人或团队独有的工作流程（如周报生成、数据分析、品牌内容创作）打包成Skill。基本步骤是：创建一个文件夹，编写核心的 `SKILL.md` 文件（包含技能描述和详细指令），并可选择性地添加脚本和资源文件，最后在网页端或通过API上传即可[citation:1][citation:7]。

### 💡 核心价值与应用场景

-   **对企业与团队**：Skills 是沉淀和标准化组织知识的利器。可以将公司的品牌规范、财务报告模板、项目文档标准等封装成技能，确保不同成员使用 Claude 时输出格式统一、质量可控的内容，极大提升效率与一致性[citation:2][citation:7]。
-   **对个人用户**：能够将重复性的工作流程（如整理会议纪要、生成特定格式的周报、批量处理图片）自动化，打造一个真正懂你工作习惯的个性化助手[citation:3]。

### ⚠️ 重要提示：安全使用

由于 Skills 可以包含并执行代码，安全至关重要。Anthropic 强烈建议用户**仅从可信来源安装技能**[citation:6][citation:7]。对于来自不熟悉来源的技能，在上传使用前应仔细审计其 `SKILL.md` 文件和相关脚本的内容，警惕任何要求连接不可信外部网络的指令，以防潜在的数据泄露或恶意操作风险[citation:1][citation:7]。
