# 元宝对话 — AI 大模型技术探索

腾讯元宝中与 AI/ML 技术相关的对话合集，共 83 轮对话。涵盖 Transformer 架构、LLM 原理、RAG、Embedding、深度学习、扩散模型、AI 编程、AI 安全等主题。

## 核心主题

1. **大模型基础架构** — Transformer 起源与原理、Encoder-Decoder 结构、Decoder-Only 架构选择、自注意力机制、位置编码等
2. **训练与微调** — LoRA 微调技术、预训练-微调范式、RLHF 三阶段训练、模型蒸馏、Scaling Laws
3. **检索增强生成 (RAG)** — Graph RAG 原理、传统 RAG 对比、知识图谱本体设计、知识图谱对大模型效果的评估
4. **Embedding 与语义表示** — Word2Vec/GloVe/FastText/BERT 发展历程、静态 vs 动态嵌入、TF-IDF/BM25 算法
5. **AI Agent 与编程** — AI Agent 发展趋势、Vibe Coding、Computer Use、AI 编程行业现状、MCP 与 Agent Skills
6. **AI 安全与趋势** — AI 泡沫论、AI 安全上海共识、生成式 AI 的世界模型缺陷、AI 裁员影响
7. **多模态与扩散模型** — Stable Diffusion 原理、字节 Seedance 视频模型、VLA 自动驾驶模型、扩散模型原理
